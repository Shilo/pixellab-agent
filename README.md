# pixellab-agent

Portable Agent Skill for routing PixelLab game-asset requests across hosted MCP, REST v2, website/editor workflows, and legacy v1.

Use it when an agent needs to choose the right PixelLab surface for characters, objects, tilesets, tiles, maps, UI, backgrounds, edits, animations, SDKs, or vibe-coding.

## Contents

- `SKILL.md` - the skill.

## Install

Copy this folder into your agent skills directory, or install it using any tool that supports the Agent Skills folder format.
